# DCL-307: Developing ReactJS Applications using TypeScript

This repository contains the lab projects and code samples created as part of the
**DCL-307 "Developing ReactJS Applications using TypeScript"** training by
[DeepCloud Labs](https://www.deepcloudlabs.com).

- Training catalog: <https://www.deepcloudlabs.com/resources>
- Catalog (PDF): <https://www.deepcloudlabs.com/resources/deepcloudlabs.catalog.for.training.and.consultancy.services-en.US-2025a.pdf>

## Topics Covered

JavaScript · TypeScript · ReactJS · React Hooks · React Reducer ·
React Context API · React Router · Redux · WebSockets · WebRTC

## Prerequisites

- [Node.js](https://nodejs.org/) (LTS recommended)
- [MongoDB](https://www.mongodb.com/) (for backend projects)
- A code editor such as [WebStorm](https://www.jetbrains.com/webstorm/) or [VS Code](https://code.visualstudio.com/)

## Getting Started

Each subdirectory is a standalone project. To run one:

```bash
cd <project-directory>
npm install
npm start        # or: npm run dev — check the project's package.json
```
